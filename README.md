### Deploy to Vercel

1.  **Create a GitHub Repository**:
    *   Go to [GitHub](https://github.com) and create a new repository.

    *   Click "Add New..." -> "Project".

    *   Vercel will automatically detect it as a static site.
    *   Click "Deploy".
